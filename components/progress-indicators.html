Components

Progress indicators

Progress indicators signal an unexpected wait time or show the length of a process to the user.

Using progress indicators

Process indicators can show determinate or indeterminate progress. They indicate an ongoing system process Links must include a descriptive link text or a combination of the text and an additional icon.

Designing

Indicator
Choosing the right indicator type is dependent on context.

Label (optional)
Adding a descriptive label is additionally helpful on indicators like a determinate progress bar. It's meant to state clearly what's being progressed on and how far system is in progressing, for example by providing a percentage number.
